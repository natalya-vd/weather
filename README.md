# CLI погоды

## Системные требования
- node.js >=16

## Установка
- Для глобальной установки запустить команду `npm i -g weather-cli-1234`

## Запуск прграммы
- через CLI: `weather`
- локально `npm start`

## Использование
- Без параметров - вывод погоды
- -s [CITY] установка города
- -t [API_KEY] сохранение токена
- -h вывод помощи
